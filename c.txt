For login in git
git config --global git.user
git config --global git.email

Commands for branch
1. For creating branch : git branch branch_name
2. For creating branch and checkout into another branch: git checkout -b branch_name
3. for switching branch : git switch branch_name
4 for deleting branch : git checkout -d branch_name
5 for deleting branch if code is not merged : git checkout -D branch_name
